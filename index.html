WHAT IS THE MAIN ELEMENT USED FOR?

THE MAIN ELEMENT IS USED TO TO SPECIFY THE MAIN CONTENT ON THE WEB PAGE

WHAT IS THE SECTION ELEMENT USED FOR?

the section tag is used for two purposes

  1. to indicate a generic or normal section on a web design page
  2. to indicate different part of the page
  3. The section element is a structural HTML element used to group together related elements.

WHAT DO WE USE THE HEADING ELEMENT FOR?

Search engines use the headings to index the structure and content of your web pages. Users often skim a page by its headings. It is important to use headings to show the document structure.
